lower_bound_set_intersection

Defined in <algorithms/triangle_count.hpp>
Scan set intersection that uses std::lower_bound for head start 
template <typename _InputIterator1, typename _InputIterator2, typename _OutputIterator>
_OutputIterator lower_bound_set_intersection(_InputIterator1 __first1, _InputIterator1 __last1, _InputIterator2 __first2, _InputIterator2 __last2, _OutputIterator __result)
(1)
  1. Compares the start index of each range, advances the beginning of the smaller index range to the beginning of the other range using std::lower_bound. Then intersect with a simple scan intersection.
    Parameters:
    • _InputIterator1 __first1 : Beginning of the first range
    • _InputIterator1 __last1 : End of the first range
    • _InputIterator2 __first2 : Beginning of the second range
    • _InputIterator2 __last2 : End of the second range
    • _OutputIterator __result : Accumulator output iterator
    Returns: The number of set intersections
