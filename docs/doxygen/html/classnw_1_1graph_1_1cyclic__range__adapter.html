iterator nw::graph::cyclic_range_adapter< Iterator >::end ( ) const
inline

Return an iterator that points to the end of the cycle.

The end of the cycle is the first iterator in the cycle that is greater than or equal to the end_ iterator in the underlying range. End iterators for different cycles will be different even if the underlying range and strides match, so tests should not be performed across cycles.
